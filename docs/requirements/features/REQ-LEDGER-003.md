# REQ-LEDGER-003 仕訳帳・総勘定元帳

<!-- review:pending id=r-20260605-004 -->

## 概要

仕訳帳（全仕訳の時系列一覧）と総勘定元帳（科目別の期首残高・明細・累計残高）を参照する。期間・科目でフィルタできる。読み取り専用（編集・削除は仕訳入力画面から行う）。

## 関連 Spec

- [REQ-LEDGER-001](./REQ-LEDGER-001.md): 仕訳入力（元帳の元データ）
- [REQ-LEDGER-002](./REQ-LEDGER-002.md): 勘定科目管理（科目フィルタ）
- [REQ-MASTER-002](./REQ-MASTER-002.md): 会計期間設定（期間フィルタ）

## アクター

- 経理担当者: 仕訳帳・総勘定元帳を参照する

## 機能要件

### 仕訳帳

仕訳帳画面（`/books/journal`）で、期間・科目フィルタを指定して仕訳一覧を表示する。

- フィルタ: 期間（from/to）、勘定科目（accountId）
- 表示項目: 日付、仕訳番号、摘要、借方科目・借方金額、貸方科目・貸方金額、ステータス
- POSTED のみ表示する（DRAFT は除外）

**受入条件:**

- [ ] 仕訳帳が期間フィルタで絞り込める
- [ ] 科目フィルタを指定すると該当科目の仕訳のみ表示される
- [ ] DRAFT ステータスの仕訳は表示されない

### 総勘定元帳

総勘定元帳画面（`/books/general-ledger`）で、指定した科目の期首残高・明細行・累計残高を表示する。

- 必須パラメータ: accountId（科目必須）
- オプション: from/to（期間）
- 期首残高: 期間開始前の累積残高
- 明細: 対象期間内の仕訳明細（日付・摘要・借方・貸方）
- 累計残高: 各明細行ごとの残高（期首残高から加減算）

**受入条件:**

- [ ] 科目を選択すると総勘定元帳が表示される（`GET /api/ledger/general-ledger?accountId=...`）
- [ ] 期首残高が表示される（期間開始前の累積値）
- [ ] 各明細行に累計残高が表示される
- [ ] 期間フィルタで表示範囲を絞り込める

## 画面

| パス | 主要UI要素 |
|------|----------|
| `/books/journal` | 期間・科目フィルタ、仕訳一覧テーブル（日付・番号・摘要・借方科目・借方金額・貸方科目・貸方金額・ステータス） |
| `/books/general-ledger` | 科目選択（必須）・期間フィルタ、期首残高行、明細テーブル（日付・摘要・借方・貸方・残高） |

## API

（出典: `interfaces/ledger_interface.md`）

| メソッド | パス | 概要 |
|---------|------|------|
| GET | /api/ledger/journal-entries | 仕訳一覧（仕訳帳として使用: ?from=&to=&accountId=） |
| GET | /api/ledger/general-ledger | 総勘定元帳（?accountId=必須&from=&to=。期首残高+明細+累計残高を返す） |

## ビジネスルール

- 集計対象は status=POSTED の仕訳のみ（DRAFT は除外）
- 総勘定元帳の accountId パラメータは必須
- 期首残高は期間開始前の累積残高（ASSET/EXPENSE は借方残、LIABILITY/EQUITY/REVENUE は貸方残）
- 累計残高は各明細行ごとに期首残高から加減算して算出する

## 受入条件（機能全体）

- [ ] 仕訳帳でPOSTED仕訳が時系列で確認できる
- [ ] 総勘定元帳で科目を指定すると期首残高・明細・累計残高が正しく表示される
- [ ] DRAFT の仕訳は仕訳帳・元帳ともに表示されない
