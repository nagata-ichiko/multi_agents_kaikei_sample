# 完了確認チェックリスト

タスク完了時に確認する共通項目。各項目はプロジェクトに該当する場合のみ確認する。

## チェック項目

- [ ] CLAUDE.md の「開発コマンド」に記載されたテストを実行し、全パスしていること
- [ ] spec-map.yml が存在する場合: 新規・変更した実装ファイルが登録されていること
- [ ] docs/api/openapi.yaml が存在し、かつAPIに変更がある場合: 変更が反映されていること
- [ ] interfaces/ にIF定義が存在し、かつドメイン間連携に変更がある場合: IF定義と実装が一致し、IFバージョンが更新されていること
- [ ] E2Eテストが存在し、かつ対象範囲に変更がある場合: E2Eテストがパスしていること
- [ ] docs/design/dependency-graph.md が存在し、かつSpecの依存関係に変更がある場合: 依存グラフが更新されていること
- [ ] spec-map.yml が存在し、かつSpecを変更した場合: 変更したREQ-IDに**依存しているSpec**（被依存先）を逆引きし、それらのSpecの再テストが必要か確認していること。再テストが必要な場合はユーザーに通知していること
- [ ] tasks/session-state.md が存在する場合: 完了したステップにチェックが入っていること
- [ ] 問題解決時にエラーリカバリーパターン（[error-recovery.md](error-recovery.md)）に該当するケースがあれば、パターンに従って対処したこと
